# Projet CYNA H3 - DevOps

Ce dépôt contient la structure DevOps du projet CYNA.

## Arborescence

- `infra/terraform/aws` : provisioning AWS
- `infra/terraform/azure` : provisioning Azure
- `infra/ansible` : configuration automatisée
- `infra/kubernetes` : manifests Kubernetes
- `infra/helm` : charts Helm
- `infra/monitoring` : supervision et observabilité
- `infra/scripts` : scripts utilitaires
- `docs` : documentation technique
- `pipelines` : fichiers CI/CD
